# Technologies used in this app
- Presentation architecture -> MVVM
- Architecture -> Clean
- Json Converter -> Gson
- HTTP Client -> Retrofit2
- Firebase -> Analytics & Crashlytics
- Database -> Room
- Dependency injection -> Dagger Hilt
- Unit Testing -> Mockito
